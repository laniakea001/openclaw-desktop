# OpenClaw Desktop 🦞

Windows 桌面端一键安装包，让用户无需命令行即可使用 OpenClaw。

## 🎯 解决的问题

- ❌ 安装复杂，命令行恐惧
- ❌ 配置繁琐，容易出错
- ❌ 环境配置问题多

**目标：让每个人都能轻松使用 OpenClaw**

## 🚀 快速开始

### 前置要求
- Windows 10/11
- 已安装 Node.js (v18+)

### 安装步骤

1. 下载最新版本
2. 运行安装程序
3. 填写 API Key
4. 选择渠道（飞书/Telegram/Discord）
5. 点击启动！

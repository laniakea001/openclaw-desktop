# OpenClaw Desktop

Windows 桌面端一键安装包，让用户无需命令行即可使用 OpenClaw。

## 功能

- ✅ 一键安装 OpenClaw
- ✅ 配置模型 API Key
- ✅ 启动/停止 Gateway
- ✅ 实时日志查看
- ✅ 飞书/Telegram/Discord 配置

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run tauri dev

# 构建
npm run tauri build
```

## 技术栈

- Tauri 2.x
- React + TypeScript
- Rust
